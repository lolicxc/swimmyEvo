**Swimmy Evolution** 
Swimmy Evolution is an endless game where you play as a mutant fish that shoots and dodges deadly sponges.

**Features**
- Single Player
-Colorful underwater graphics
-Collage-style art inspired by marine life

**Controls**
-SPACE for jump
-MOUSE LEFT BUTTON for shooting

**Install instructions**
-Download the Swimmy Evolution ZIP file.
-Extract the contents of the ZIP to any folder on your computer.
-Open the folder and run SwimmyEvo.exe.
-Play and enjoy!
